# Pallavi P V Portfolio

Professional React portfolio built from the provided resume and photo.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually `http://localhost:5173`.

## Edit Content

- Main portfolio content: `src/main.jsx`
- Styling: `src/styles.css`
- Profile photo: `src/assets/pallavi-profile.jpeg`
- Downloadable resume: `public/Pallavi_PV-Resume.pdf`

## Build

```bash
npm run build
```
